# Sample Data Report Preview

This is a sample preview document for testing the data report functionality.

## Overview
This document provides an overview of the employee data that will be analyzed.

## Data Description
The dataset contains employee information including:
- Name: Employee full name
- Age: Employee age in years
- City: Employee location
- Salary: Annual salary in USD

## Expected Analysis
The analysis should include:
1. Salary distribution by city
2. Age demographics
3. Average salary by location
4. Correlation between age and salary

This preview document helps understand the context and expected outcomes of the data analysis.
